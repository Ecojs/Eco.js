get

  • get(dataset: string, dayStart?: number, dayEnd?: number): Promise<StatReturn>

  • Returns 100 data points between dayStart and dayEnd of the selected data. If there are more than 100 samples, it will be averaged out to contain exactly 100.

    Parameters

    • dataset: string

      dataset to take the data from.

    • OptionaldayStart: number = 0

      Day from which on data is returned. Default is 0.

    • OptionaldayEnd: number = -1

      Day until which data is returned. Default is now.

    Returns Promise<StatReturn>

    A list of floats that returns the data. Might be averaged out if there were too many data points.
